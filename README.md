This repository contains practical, battle-tested guides and step-by-step instructions for Exchange Server administrators.

All documentation is organized inside the [`docs/`](./docs/) folder for better readability and navigation.

## How to Use

- Start with the [DAG Maintenance Procedures](./docs/DAG-Maintenance.md) guide.
- All PowerShell commands have been tested on **Exchange Server 2016** and **2019**.
